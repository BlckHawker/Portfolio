KTANE Projects

Keep Talking and Nobody Explodes is a co-op game where all parties must work together in order to defuse a bomb. The bomb is made up of puzzles called “modules”. One person takes the role of the defuser. They can interact with the bomb. Everyone else is an expert; they have access to the manuals that tell how to defuse the modules. Defusers are not allowed to look at the manuals while the experts are not allowed to look at the bomb. Both parties must communicate with each other in order to defuse the bomb. Each page of the manual tells how to solve a specific module given some information from the bomb. Solving all modules before time runs out will defuse the bomb. The modding community flourished in 2016 where players could create their own modules and other tools to enhance the experience playing the game. Here are some of my contributions to the community.

Log File Analyzer (LFA)

July 2023 - Present

Tools: Regex, Javscript, Jquery, CSS, SVG, Visual Studio Code

The log file analyzer is a tool created by the community that parses logs printed from modules in a more digestible way. Most logs tell the user how to solve the module and the steps to get there. Mods are limited in what they can log since they can only use text. LFA has the ability to add svg graphics, put messages into dropdowns and code blocks, reword logging to make it more clear, and much more. Here are my contributions to LFA.
